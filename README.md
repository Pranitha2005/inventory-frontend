# Inventory Monitoring System — Frontend

## 📌 Overview
A full-stack Inventory Monitoring System built with React.js and Firebase. 
Helps businesses track stock levels in real time with automatic low stock alerts.

## 🚀 Live Features
- ➕ Add / Delete products
- ⚠️ Real-time Low Stock Alerts (quantity < 10)
- ✅ In Stock / Low Stock status badges
- 🔥 Firebase Realtime Database — data persists permanently
- 📊 Inventory table with category, price, quantity

## 🛠️ Tech Stack
- React.js
- Firebase Realtime Database
- Axios
- CSS-in-JS styling

## 📁 Related Repository
- Backend: [inventory-backend](https://github.com/Pranitha2005/inventory-backend)

## ▶️ How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000`
